## 2. Technical Decisions & Dependencies
* **Midpoint Centering Formula:** To perfectly center an absolute element inside a gap above a relative parent:
  1. Set a defined top margin on the parent wrapper (e.g., `32px`).
  2. Set the absolute element's `top` to exactly `-1/2` of that margin (e.g., `-16px`).
  3. Use `transform: translate(-50%, -50%)` to shift the element's exact geometric center to that coordinate.
