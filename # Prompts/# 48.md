## 2. Technical Decisions & Dependencies
* **Data-Driven Actions:** We are decoupling step offsets from the UI. The store will hold a `stepperConfig` array of exactly 24 `StepperAction` objects.
* **XOR Inversion Logic:** The engine requires global vector inversion.
  * `isInverted = uiState.stepperInvertToggle !== uiState.stepperInvertMomentary`
  * When inverted, all numeric step/octave values are multiplied by `-1` before execution.
* **Store-Level Note Tracking:** To allow the UI to remain a "dumb renderer", the Store will take over the responsibility of mapping a physical key index to its generated output MIDI note, so it can reliably trigger `removeOutputKey` on NoteOff.

### Phase 2: Action Routing Logic — ✅ Completed
* **Objective:** Implement the `triggerStepperAction` function in the store to process these keys.
* **Instructions:**

Import `executeScaleStep` into `useMidiStore.ts` (if needed) or pass the calculated step out. *Architect Note:* To avoid circular dependencies, implement a pure store action: `processStepperAction: (index: number, isNoteOn: boolean, executeEngineFn?: (offset: number) => number | null) => void`.
  2. **Logic Tree for NoteOn (`isNoteOn === true`):**
     * Get `action = stepperConfig[index]`.
     * Add `48 + index` to `activeKeys`.
     * **If INVERT_TOGGLE:** Toggle `stepperInvertToggle`.
     * **If INVERT_MOMENTARY:** Set `stepperInvertMomentary` to true.
     * **If HOME:** Call `get().triggerHomeReset()`.
     * **If STEP / OCTAVE / REPEAT_LAST:**
       * Resolve effective action (if REPEAT_LAST, use `lastStepperAction`. If null, abort).
       * If OCTAVE, the base step is `effective.value * currentScale.length` (derive length from `getLUT()[scaleDecimalId].pitch_class_set.length`).
       * Apply XOR inversion: If `toggle !== momentary`, multiply the base step by `-1`.
       * If `executeEngineFn` is provided, call it with the final step value. If it returns a valid MIDI number, save it to `stepperActiveNotes[index]`. Update `lastStepperAction` (unless this was a REPEAT_LAST call).
  3. **Logic Tree for NoteOff (`isNoteOn === false`):**
     * Remove `48 + index` from `activeKeys`.
     * **If INVERT_MOMENTARY:** Set `stepperInvertMomentary` to false.
     * Check `stepperActiveNotes[index]`. If a note exists, call `removeOutputKey(note)` and delete the entry from `stepperActiveNotes`.
