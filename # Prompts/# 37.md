# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
This task migrates the "On Scale Change Behavior" preference out of the global settings modal and attaches it directly to the `KeySwitchContainer`, which is the component it primarily affects. We will change the default setting to "Follow Root", remove it from the global `SettingsModal.tsx`, create a new dedicated `ScaleChangeSettingsModal.tsx` (styled exactly like the `PlayStartSettingsModal.tsx`), and add a cog icon to the Key Switch info strip to trigger it. 

## 2. Technical Decisions & Dependencies
* **Component Architecture:** A new file `src/components/ScaleChangeSettingsModal.tsx` must be created.
* **Styling Strictness:** The wrapper, backdrop blur, exit (`X`) button, and exact CSS classes of `PlayStartSettingsModal.tsx` must be used. However, the inner control will be the 2-button split toggle (previously in `SettingsModal.tsx`), with "Follow Root" ordered first.
* **Zustand Defaults:** The default state for `scaleChangeMode` in `useMidiStore.ts` must be flipped.
