# Product Requirements Prompt (PRP) — 🔘 Pending

## 1. Project Context & Objectives
**Project:** MIDI Scale Stepper MVP — **Job 4: Scale Select Symmetry & State Sync**
**Objective:** The Stepper and Output engines are complete. However, the "Scale Select" zone (MIDI C3-B3 / 48-59) and the `ScaleKeySwitches12.tsx` component are out of sync. Currently, MIDI input mutates the text of the first key switch instead of changing the active selection pointer. 

We must lift the `selectedIndex` into Zustand so the MIDI hardware and the UI share the exact same active pointer, and we must remove the side-effect mutations.

## 2. Implementation Phases

* **Objective:** Create a single source of truth for which of the 12 Scale Switches is currently active.
* **Instructions:**

* **Instructions:**
