# Product Requirements Prompt (PRP) — Scale Change Logic & Math Fix 🔘

## 1. Project Context & Objectives
The "On Scale Change" logic is currently experiencing a mathematical bug causing the Start Note to infinitely drift downwards into lower octaves. Furthermore, the "Follow Root" UI setting is being ignored because components are bypassing the store's built-in setter actions and mutating state directly via `setState`.

We must fix the negative modulo bug in `computeNewLastPlayedMidi`, add diagnostic console logs, and audit the codebase to ensure all scale and root changes are routed through `setActiveState` or `setRootNote`.

## 2. Technical Decisions & Dependencies
* **Positive Wrapping Modulo:** Javascript's `%` operator fails on negative numbers. In `computeNewLastPlayedMidi`, the Voice-Leading array map must be updated to:
  `const absolutePitchClasses = pitch_classes.map((pc: number) => (((root + pc) % 12) + 12) % 12);`
* **Direct State Mutation Ban:** No component (`ScaleKeySwitches12`, `ScaleInspectorNotation`, `useWebMidi`) is allowed to use `useMidiStore.setState(state => ...)` to change the active scale or root note. They MUST use `setActiveState`, `setRootNote`, or `setScaleDecimalId` so the `computeNewLastPlayedMidi` engine is triggered.

      const pref = state.globalSettings.roundPreference;
      const root = newRootNote !== undefined ? newRootNote : state.activeState.rootNote;
      const startingOctave = state.globalSettings.startOctave;

      if (mode === 'voice-leading') {
        if (lastPlayed !== null && decimal !== null) {
          const lut = getLUT();
          const entry = lut ? lut[decimal] : null;
          const pitch_classes = entry ? entry.pitch_class_set : [0, 2, 4, 5, 7, 9, 11];
          
          // STRICT FIX: Positive wrapping modulo to prevent downward octave drift

          return newNote;
        }
        return lastPlayed;
      } else {
        // Follow Root
        const pc = (((root ?? 0) % 12) + 12) % 12;
        const homeNote = pc + ((startingOctave + 1) * 12);

      }
