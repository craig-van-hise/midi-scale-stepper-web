# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
**UI REFINEMENT:** The absolute positioning of the Octave Knob above the "PLAY/START NOTE" banner is currently using a hardcoded `top` offset that pushes it too close to the card's top border, destroying vertical symmetry.

**Objective:** Refactor the CSS positioning of the Octave Knob so it anchors cleanly to the top edge of the cyan banner with a strict, symmetrical bottom margin, ensuring it sits perfectly vertically centered in the negative space.

## 2. Technical Decisions & Dependencies
* **Bottom Anchoring:** Instead of using `top: '-45px'`, the absolute container will use `bottom: '100%'` coupled with a small `marginBottom`. This mathematically guarantees the knob will always sit exactly above the banner without overlapping it, regardless of screen scaling.
* **Horizontal Centering:** The `left: '50%', transform: 'translateX(-50%)'` logic remains the correct approach for strict horizontal symmetry relative to the cyan banner.
