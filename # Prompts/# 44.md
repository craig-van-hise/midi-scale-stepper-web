# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
This PRP resolves two critical bugs in the Play/Start Note zone:
1. **The Filter Trap:** If a user plays a note that is dropped by the Output Filter, the Play/Start zone fails to update the global `lastPlayedMidi` anchor, causing the UI to flash like a momentary switch instead of acting like a sticky configuration key.
2. **The Visual Octave Desync:** When the Stepper changes the `lastPlayedMidi`, the Play/Start UI attempts to highlight that exact note without applying the reverse `octaveOffset`, resulting in incorrect or out-of-range visual highlighting.
3. **Default State:** The `octaveOffset` must be defaulted to `-2`.

## 2. Technical Decisions & Dependencies
* **State Extrication:** The `setLastPlayedMidi` and `activeNotesRegistry` calls must be moved *above* and *outside* of the `applyOutputFilter` logic in both the physical MIDI handler and the on-screen keyboard handler. The anchor must update regardless of audio muting.
* **Reverse Mapping Math:** To highlight the correct physical key, the UI loop must subtract the offset: `lastPlayedMidi - (octaveOffset * 12)`.

### Phase 2: Fix The Filter Trap (State Extrication) — ✅ Completed
* **Objective:** Ensure the anchor state updates before output filtering drops the note.
* **Instructions:**

    2.  Refactor the logic to strictly follow this order:
        ```typescript
        const rawNote = Math.max(0, Math.min(127, note + ((octaveOffset ?? 0) * 12)));
        const roundedNote = roundNote(rawNote, state.activeState.scaleDecimalId, state.globalSettings.roundPreference);
        const targetNote = rounded ? roundedNote : rawNote;
        
        // 1. ALWAYS set anchor and registry FIRST
        state.setLastPlayedMidi(targetNote);
        activeNotesRegistry.current.set(note, targetNote);
        
        // 2. Apply Output Filter
        const finalTargetNote = applyOutputFilter(targetNote, filterMode, filterRange[0], filterRange[1]);
        
        // 3. Route to Output
        if (finalTargetNote !== null && audible) {
          state.addOutputKey(finalTargetNote);
        }
        
        state.addActiveKey(note);
        return;
        ```
